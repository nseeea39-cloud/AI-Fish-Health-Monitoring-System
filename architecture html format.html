AI Fish Health Monitoring System

System Architecture Pipeline

End-to-end computer vision pipeline: from underwater camera frames to Power BI health dashboards, powered by Roboflow & Python.

Step 3 — Processing
Python Inference
Python script orchestrates real-time frame capture, calls Roboflow SDK, classifies each detection as Healthy or Diseased, draws bounding boxes, and writes structured log records.
roboflow SDK inference OpenCV · real-time pandas datetime

📋
Step 4 — Storage
CSV Logs
Each detection event is appended to a daily-rotated CSV file. Columns include timestamp, class (Healthy / Diseased), confidence score, and bounding-box coordinates.
timestamp class_label confidence x · y · w · h daily rotation
CSV import / refresh

📈
Step 5 — Visualisation
Power BI Dashboard
Scheduled refresh pulls the CSV logs into Power BI. Dashboards show health scores, Healthy vs. Diseased trends over time, confidence distributions, and per-camera KPIs.
Power Query DAX measures Health Score KPI trend charts scheduled refresh
Detection Classes
Two-Class Classification
The model outputs exactly one label per detected fish
Healthy
Fish exhibiting normal swimming behaviour, clear eyes, intact fins, and natural colouration. No visible lesions or abnormalities detected.
normal behaviour clear eyes intact fins natural colour
avg. confidence 92%
Diseased
Fish displaying signs of illness: abnormal swimming, discolouration, lesions, fin rot, cloudy eyes, or erratic movement patterns.
lesions fin rot discolouration cloudy eyes
avg. confidence 88%
