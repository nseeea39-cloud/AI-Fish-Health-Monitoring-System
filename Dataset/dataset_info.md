# Dataset Information

## Fish Health Dataset

### Classes

1. Healthy
2. Diseased

### Original Dataset

- Healthy Images: 479
- Diseased Images: 843

### After Augmentation

- Total Images: 2268

### Dataset Split

- Training Set: 1892 Images (83%)
- Validation Set: 254 Images (11%)
- Test Set: 122 Images (5%)

### Annotation Type

Single Label Image Classification

### Purpose

The dataset is used to train a ResNet50 classification model to identify healthy and diseased fish based on visible skin conditions.
